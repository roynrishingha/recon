<h1 align="center">recon</h1>

## Usage

1. clone locally

```sh
git clone https://github.com/royrustdev/recon.git
cd recon
```

2. compile

```sh
cargo build --release
```

3. run

```sh
./target/release/recon <example.com>
```

### Example

```sh
./target/release/recon royrustdev.com
```

or

```sh
cargo run --release -- royrustdev.com
```
